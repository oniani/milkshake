# milkshake

Milkshake: A Space Odyssey

## Keybindings

| Keybinding       | Description      |
| ---------------- | ---------------- |
| <kbd>→</kbd>     | Move Right       |
| <kbd>←</kbd>     | Move Left        |
| <kbd>↑</kbd>     | Jump             |
| <kbd>SPACE</kbd> | Shoot Laser Beam |
| <kbd>M</kbd>     | Show Help        |
